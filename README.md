# いえ電シェア（family-electricity-cost-share）

家庭内で使用した冷暖房機器（エアコン・ストーブ等）の利用時間を記録し、電気料金を公平にシェアするための Web アプリケーションです。

これまで手書きノートと Excel で管理していた冷暖房の利用時間と電気料金の集計を、より正確かつ簡単に行うためのツールとして開発しました。

---

# アプリ概要

家庭内で複数人が冷暖房機器を利用した場合、

- 誰が
- どの機器を
- どれくらい使用したか

を記録し、電力会社の請求額をもとに  
**各メンバーの負担金額（シェア金額）を自動計算**します。

---

# 主な機能

## 一般ユーザー

- 機器の運転開始 / 停止
- 機器使用記録の一覧表示
- 記録の編集 / 論理削除
- シェア金額の確認
- シェア金額の内訳表示  
  - 機器使用分  
  - 割り勘分  
  - シェア金額

---

## 管理者

- シェアメンバー管理
- 機器管理
- 仮単価設定
- 電気料金確定処理
- 確定履歴一覧 / 履歴詳細
- 論理削除された記録の復旧

---

# 電気料金計算ロジック

### ① 機器使用分


機器消費電力 × 使用時間 × 単価


### ② 割り勘分


(請求総額 − 機器使用料金合計) ÷ シェアメンバー人数


※機器を使っていないメンバーも割り勘対象

### ③ シェア金額


機器使用分 + 割り勘分
